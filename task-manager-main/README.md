# task-manager

## Описание

task manager - это веб-сайт, созданный с использованием фреймворка Django. Он предоставляет собой сервис для создания задач с возможностью указания приоритета, срока задачи и  выполнения.

## Установка

1. **Скачайте файлы:**

2. **Активация виртуальной среды:**

После создания виртуальной среды активируйте ее в вашем терминале:

- На Windows:

  ```
  myenv\Scripts\activate
  ```

- На macOS и Linux:

  ```
  source myenv/bin/activate
  ```

3. **Установка зависимостей:**

  Убедитесь, что ваша виртуальная среда активирована, и установите зависимости из файла `requirements.txt`:

  
  ```
  pip install -r requirements.txt
  ```

4. **Создайте миграции:**

  
  ```
  python manage.py makemigrations
  ```

5. **Примините миграции:**

   
  ```
  python manage.py migrate
  ```
   
6. **Запустите сервер:**
   
   
  ```
  python manage.py runserver
  ```
